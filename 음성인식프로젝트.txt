0B0E-0B11 전투 중인 포켓몬의 기술 임시데이터


음성인식 : 카카오 API / 크롬으로 녹음을 시작하고 끝난걸 체크하고 결과 텍스트 추출
파이썬에서 결과 텍스트에 대응하는 기술번호를 엑셀표에서 찾아 변환, 결과 전송용 루아 스크립트에 실행 신호(a=1)와 기술 번호(hex = xxx) 변수 값으로 저장

루아 스크립트(Voice to Moves.lua): 프레임마다 전송용 스크립트에서 실행 신호가 왔는지 확인하고 실행 신호 수신시 기술 번호를 전투 중인 포켓몬의 모든 기술에 입력, A버튼으로 강제 기술실행

0.5초 뒤 파이썬에서 전송용 스크립트의 실행 신호를 다시 0으로 되돌림
